# pi-extension-backbone-client

Pi extension that exposes todo/reminder tools by calling the backbone API.

## Env

- `BACKBONE_API_BASE_URL`
- `BACKBONE_API_TOKEN`
- `DEFAULT_TIMEZONE`
- `TODO_USER_ID` (optional default; otherwise pass `userExternalId` to every tool)

## Build

```bash
npm install
npm run typecheck
```
